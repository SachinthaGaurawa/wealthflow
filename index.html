📲 Install WealthFlow App
  1. Open this site in Chrome or Edge.
  2. Tap the ⋮ menu (top-right).
  3. Choose "Install app" or "Add to Home Screen".
  4. Confirm — WealthFlow will appear on your home screen as a real app icon.
💡 If you saw an "Install App" button at the top of the browser, tap it instead — it's the fastest way.
  1. Open this site in Safari (not Chrome — only Safari supports installing).
  2. Tap the Share button (square with arrow up) at the bottom.
  3. Scroll down and tap "Add to Home Screen".
  4. Tap Add in the top-right corner.
⚠️ iOS does not show automatic install prompts. The Share-sheet route is the only way Apple permits.
  1. Open this site in Chrome, Edge, or Brave.
  2. Look for the install icon (⊞) in the right side of the address bar.
  3. Click it and choose "Install".
  4. WealthFlow will open as a standalone window and pin to your taskbar / Start menu.
💡 Once installed, you can launch it like any other Windows app.
  1. Open this site in Chrome or Edge (Safari install limited).
  2. Click File → Install WealthFlow or the install icon (⊞) in the address bar.
  3. Confirm — the app will appear in your Applications folder and Launchpad.
💡 You can also drag from /Applications to your Dock for one-click access.
